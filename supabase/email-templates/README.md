# Notluk Supabase E-posta Şablonları

## Magic Link or OTP

Konu:

```text
Notluk giriş kodun
```

Gövde:

`magic-link-or-otp.html` dosyasını kullanın.

Gönderen adı:

```text
Notluk
```

Supabase hosted projelerde e-posta şablonunu ve gönderen bilgisini değiştirmek için önce Custom SMTP ister. SMTP etkinleştirildikten sonra bu şablonu Authentication -> Emails -> Templates -> Magic Link or OTP alanına yapıştırın.
